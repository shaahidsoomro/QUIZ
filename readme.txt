A comprehensive WordPress plugin to manage multiple choice questions (MCQs), exams with timers, user dashboards, quiz analytics, import/export, and more — built by PAKSTUDY.XYZ

== Description ==

Pak Study Quiz Manager is an advanced MCQ & exam plugin tailored for educators, students, and quiz admins. Ideal for online testing and learning environments.

**Key Features:**
- MCQ Management (Statement, 4 options, correct answer, category, difficulty, explanation)
- Exam Manager (100 MCQs, countdown timer, login required, disclaimer popup)
- User Dashboard (Quiz attempts, scores, accuracy tracking)
- Admin Dashboard with Charts (MCQ stats, exam stats, leaderboard)
- One-MCQ-Per-Page with forced popup for answer reveal
- AJAX-based live quiz interaction
- CSV Import/Export of MCQs and Exams
- Email notifications for registration, quiz attempts, and abandoned exams
- Shortcodes for quizzes and exams
- Fully responsive UI with clean modals and dashboards
- Icons for each admin menu (MCQs, Exams, Analytics, Dashboard)

**Shortcodes:**
- `[mcq_quiz]` – Display one MCQ per page quiz with forced popup answer
- `[exam_list]` – Public exam listing (logged-in users only)
- `[exam_result]` – Exam score/results for logged-in users
- `[mcq_user_dashboard]` – User dashboard for quiz/exam tracking

== Installation ==

1. Upload the plugin folder `pakstudy-quiz-manager` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the admin menu “MCQs” to add or import MCQs and create exams.
4. Use shortcodes on pages to display quizzes and exams.

== Frequently Asked Questions ==

= How many MCQs can I add? =
There is no hard limit. The plugin is optimized for thousands of MCQs with import/export support.

= Can I force users to see correct answers? =
Yes. The plugin uses a popup to compel users to see the correct answer before proceeding.

= Can I limit exams to logged-in users only? =
Yes. Exams are restricted to registered and logged-in users.
